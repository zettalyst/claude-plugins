# Cancel Clarify Ralph Loop

Cancel the active clarification loop:

```!
rm -f .claude/clarify-ralph.local.md && echo "Clarify Ralph loop cancelled."
```

The clarification loop has been cancelled. You can now exit the session normally or start a new loop with `/clarify-ralph`.
